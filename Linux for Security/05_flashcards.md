# Flashcards: Linux for Security

## How to use
Cover the answer column and test your memory.

| Question / Concept | Answer |
| :--- | :--- |
| **UID of the root user** | 0 |
| **Principle: Give minimum necessary access** | Least Privilege |
| **Default SSH Port** | 22 |
| **Command to secure SSH: Disable this login** | `PermitRootLogin no` |
| **Tool to ban IPs after failed logins** | `fail2ban` |
| **Uncomplicated Firewall command** | `ufw` |
| **UFW: Command to allow port 80** | `ufw allow 80` |
| **Log file for SSH/Sudo events** | `/var/log/auth.log` |
| **Command to see who logged in before** | `last` |
| **Command to see failed login attempts** | `lastb` |
