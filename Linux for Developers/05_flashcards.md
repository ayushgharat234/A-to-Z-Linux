# Flashcards: Linux for Developers

## How to use
Cover the answer column and test your memory.

| Question / Concept | Answer |
| :--- | :--- |
| **Package group for C compilers** | `build-essential` |
| **Tool to manage Node.js versions** | `nvm` |
| **Command to create Python venv** | `python3 -m venv name` |
| **Git: Initialize repo** | `git init` |
| **Git: Stage changes** | `git add` |
| **Git: Save changes** | `git commit` |
| **Git: Create new branch** | `git switch -c name` |
| **File to exclude from Git** | `.gitignore` |
| **Docker: Read-only template** | Image |
| **Docker: Running instance** | Container |
| **Command to run a container** | `docker run` |
| **Command to list running containers** | `docker ps` |
| **Command to build an image** | `docker build` |
| **Flag to run Docker in background** | `-d` |
| **Flag to map ports** | `-p` |
