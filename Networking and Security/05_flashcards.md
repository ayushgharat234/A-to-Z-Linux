# Flashcards: Networking & Security

## How to use
Cover the answer column and test your memory.

| Question / Concept | Answer |
| :--- | :--- |
| **Command to show IP address** | `ip addr` |
| **Command to test connectivity** | `ping` |
| **Command to fetch web page text** | `curl` |
| **Command to show open ports** | `ss -tulpn` or `netstat` |
| **Port: SSH** | 22 |
| **Port: HTTP** | 80 |
| **Port: HTTPS** | 443 |
| **File: Local hostname mapping** | `/etc/hosts` |
| **File: DNS nameservers** | `/etc/resolv.conf` |
| **Command to generate SSH keys** | `ssh-keygen` |
| **Command to copy SSH ID to server** | `ssh-copy-id` |
| **SCP: Upload example** | `scp local remote:` |
| **Rsync flag: Archive mode** | `-a` |
| **Rsync flag: Verbose** | `-v` |
| **Rsync flag: Compress** | `-z` |
