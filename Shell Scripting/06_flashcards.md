# Flashcards: Shell Scripting

## How to use
Cover the answer column and test your memory.

| Question / Concept | Answer |
| :--- | :--- |
| **Shebang line** | `#!/bin/bash` |
| **Variable assignment syntax** | `var=value` (No spaces) |
| **Access a variable** | `$var` or `${var}` |
| **First script argument** | `$1` |
| **All script arguments** | `$@` |
| **CLI Input command** | `read` |
| **Exit code for success** | 0 |
| **Variable for last exit code** | `$?` |
| **Comparison: Equal (numbers)** | `-eq` |
| **Comparison: Equal (strings)** | `=` |
| **Check if file exists** | `-f filename` |
| **Redirect Output (overwrite)** | `>` |
| **Redirect Output (append)** | `>>` |
| **Pipe operator** | `|` |
| **Grep: Case insensitive flag** | `-i` |
| **Sed: Replace text** | `sed 's/old/new/'` |
| **Awk: Print first column** | `awk '{print $1}'` |
| **Run script in debug mode** | `bash -x script.sh` |
