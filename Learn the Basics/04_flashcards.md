# Flashcards: Learn the Basics

## How to use
Cover the answer column and test your memory.

| Question / Concept | Answer |
| :--- | :--- |
| **Command to list files** | `ls` |
| **Command to find current directory** | `pwd` |
| **Command to change directory** | `cd` |
| **Symbol for home directory** | `~` |
| **Command to create a directory** | `mkdir` |
| **Command to copy a file** | `cp source dest` |
| **Command to move or rename a file** | `mv old new` |
| **Command to remove a file** | `rm file` |
| **Command to view file content** | `cat file` |
| **Command to view large files page-by-page** | `less file` |
| **Editor: Save in Nano** | `Ctrl + O` |
| **Editor: Exit Nano** | `Ctrl + X` |
| **Editor: Modes in Vim** | Normal, Insert, Command |
| **Vim: Save and Quit** | `:wq` |
| **Permission: Read value (octal)** | 4 |
| **Permission: Write value (octal)** | 2 |
| **Permission: Execute value (octal)** | 1 |
| **Command to change permissions** | `chmod` |
| **Command to change ownership** | `chown` |
| **Command to search for files by name** | `find . -name "filename"` |
| **Command to search text inside files** | `grep "text" file` |
| **Command to create a tar archive** | `tar -cvf archive.tar folder/` |
| **Command to extract a tar archive** | `tar -xvf archive.tar` |
